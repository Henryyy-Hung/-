## **制作目的**
* 提供一款真正简单易用的小说下载器。
  * 仅需复制粘贴网址，点击开始按钮。
* 方便使用封闭式阅读器的朋友以及需要离线阅读的中小学生追书。
  * 一经建立书单，后续一键下载最新章节TXT。
* 提供排版过的原始小说文本。
  * 排版格式为：标题不缩进，段落缩进两个中文全角空格，章节间空两行。同时自动去除网址等广告。
* 方便分享书籍。
  * 只要将 `book_info.html` 文件分享给别人，别人就能够使用你的书单。

## **使用方法**
1. 从【[**支援的网站**](#支援的网站 "Goto 支援的网站")】中选择一本你中意的书，点开目录页面。（请勿使用其它网站！！！）
&embp;例如：https://www.uuks.org/b/52716/
2. 复制目录页面的网页链接。
  * 例如：复制以上链接
3. 回到应用，手动输入书名至【**书名栏**】，并复制链接至【**链接栏**】。
  * 例如：【书名：道诡异仙】【链接：[https://www.uuks.org/b/52716/](https://www.uuks.org/b/52716/)】
4. 点击右侧【**添加数据**】按钮，将书籍放入候选区。
  * - [ ] `道诡异仙` <- 候选区状态
5. 在候选区勾选书籍，点击【**开始下载**】，即可开始下载。
  * - [X] `道诡异仙` <- 候选区状态
6. 下载完成后，点击【**查看文件**】按钮，打开装有已下载文件的文件夹。
  * 你会在文件夹里见到 `道诡异仙.txt`
7. 如有再次下载的需要，请点击【**保存书单**】，以便下次打开时恢复数据。
  * 注：应用程序同一目录下名为 `book_info.html` 的文件是用户自己创建的书单，请勿删除。
8. 其他操作：
  * 右键单击选项：编辑选项
  * 右键双击选项：用浏览器打开选项的网址
